# MyContra
Contra by cocos2d 3.17

一个基于Cocos2dx 3.17引擎制作的魂斗罗类游戏,怪物是马里奥的素材  
本人写了这东西是纯属C++课程设计作业,有需要的可以拿去用

实现了几乎所有原版魂斗罗的功能，包括平台跳跃碰撞以及怪物的生产和关底判定
